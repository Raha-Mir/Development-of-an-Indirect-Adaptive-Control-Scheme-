What does this do?

Imagine a robot arm that doesn't know its own weight, or how heavy the object it's holding is. This project teaches the arm to figure that out on the fly — while still moving to the right place, smoothly and accurately.

1

Measure position only

Only joint angles are read. Velocity must be estimated — that's the observer's job.

2

Estimate unknown parameters

The RLS algorithm watches the arm move and updates its model of mass, inertia, and friction in real time.

3

Control with estimated knowledge

The controller uses these estimates to compute the exact torques needed — even as conditions change.

Core components

State Feedback Controller

Uses inverse dynamics to cancel nonlinear effects, reducing the problem to a simple linear system.

High-Gain Observer (HGO)

Estimates joint velocities from noisy position measurements only — no velocity sensor needed.

RLS Parameter Estimation

Recursive least squares with forgetting factor tracks changing dynamic parameters in real time.

Lyapunov Stability

Mathematically verifies semi-globally practically exponential stability of the closed-loop system.

Mathematical model

The manipulator dynamics follow the Euler-Lagrange formulation. The controller uses estimated versions of each term:

Manipulator dynamics
τ = M(q)q̈ + C(q,q̇)q̇ + G(q)
Adaptive control law (with observer state ξ̂)
τ = Ĉ(q,ξ̂₂)ξ̂₂ + Ĝ(q) + M̂(q)[q̈_ref − Kₚq̃ − Kd(ξ̂₂ − q̇_ref)]
RLS update (θ̂ = parameter vector, Φ = regressor)
θ̂̇(t) = P(t)Φ̂(t)[τ(t) − Φ̂ᵀ(t)θ̂(t)]

UR5e (6-DOF) in MuJoCo 3.8.1. Quintic trajectory from rest q₀=[0,0,0,0,0,0] → qf=[0,−1.57,1.57,1.57,1.57,0] rad. Error converges to zero within ~4 s.

λ=1.005 · p₀=10⁵I · ε=0.01 · κ₁=1 · κ₂=2 · Kₚ=diag(400×6) · Kd=diag(40×6)

Periodic joint-space circular motion in the (q₁,q₂) plane — radius 0.3 rad at ω=2π/5 rad/s, continuous tracking.

q₁(t) = 0.3·sin(ωt)  ·  q₂(t) = −1.57 + 0.3(cos(ωt)−1)

3D Cartesian helical path — fully coupled motion across all 6 joints simultaneously, with descending Z.

x=xc+r·cos(ωt)  ·  y=yc+r·sin(ωt)  ·  z=z₀−(b/2π)ωt
r=0.1m · b=0.05m/rev · ω=2π/15 rad/s
